## 📸 关于 MemCull

MemCull 是一款专注于**高效、唤起回忆式的清理**手机相册的工具。我们认为整理相册不应是一项枯燥的任务，而是一次重温美好瞬间的旅程。

### 核心功能

- **极简滑动交互**：采用直观的“上滑删除，下滑保留”逻辑，结合流畅动画与震动反馈，让清理过程快如闪电。
- **深度 EXIF 解析**：实时展示照片的快门、光圈、ISO、焦距及镜头型号等专业参数。
- **高精度地理位置**：集成高德地图 LBS 服务，精准还原照片拍摄地的详细位置信息。
- **持久化回收站**：内置数据持久化回收站，支持误删找回，即便重启应用也不会丢失。
- **隐私至上**：所有图片均在本地处理，通过高级权限（如 MANAGE_EXTERNAL_STORAGE）减少系统弹窗干扰，专注清理本身。
- **美学设计**：采用 Apple 风格的 Cupertino UI 设计，极致的磨砂玻璃视觉体验。
